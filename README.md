Run Server
==========

If the file 'ping.py' is not executable,
use the command

    chmod u+x ./ping.py

To run the server use the command

    ./ping.py -s localhost


Run Ping
========

With PYTHON
-----------

To ping the server run

    ./ping.py [server-address]

With C++
--------

First compile the program, run

    make

To ping the server run

    ./ping [server-address]

or run

    ./ping [server-address] [server-port]
